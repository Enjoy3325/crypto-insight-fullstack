# Crypto Insight Dashboard

## Що це?
Повнофункціональна платформа для моніторингу криптовалют з real-time оновленнями.

## Технології
- Frontend: Next.js 14, TypeScript, Tailwind CSS
- Backend: NestJS, MongoDB, Redis
- Real-time: Socket.io
- Auth: JWT + Email OTP
